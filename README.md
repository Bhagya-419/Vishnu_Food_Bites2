# 🍽️ Vishnu Food Bites

A simple and user-friendly **Food Ordering Website** designed for college canteens/hostels, where students can browse menus, place orders, and manage their food easily.
This project focuses on **clean design, easy navigation, and efficient backend integration**.

---

## 🚀 Features

### 👤 User Side

* 📋 Menu display with categories and prices
* 🛒 Add to cart and manage items
* 🔑 User authentication (Login/Signup)
* 💳 Place orders
* 📊 View order history

### 🔐 Admin Side

* Manage food items
* Accept/reject orders
* View all user orders

---

## 🛠️ Tech Stack

### Frontend

* HTML5
* CSS3
* JavaScript

### Backend & Database

* PHP
* MySQL (XAMPP / phpMyAdmin)

## ⚙️ Setup & Installation

### 1️⃣ Clone Repository

```bash
git clone https://github.com/Bhagya-419/Vishnu_Food_Bites2.git
cd Vishnu_Food_Bites2
```

---

### 2️⃣ Setup Instructions

1. Install **XAMPP** and start:

   * Apache
   * MySQL

2. Open **phpMyAdmin**

   * Create database: `vishnu_food_bites`

3. Import:

   * `vishnu_food_bites.sql`

4. Move project folder to:

```
C:\xampp\htdocs\VISHNU_FOOD_BITES2\
```

5. Configure database (`db_config.php`):

```php
$servername = "localhost";
$username = "root";
$password = "";
$database = "vishnu_food_bites";
```

6. Run in browser:

```
http://localhost/VISHNU_FOOD_BITES2/
```
